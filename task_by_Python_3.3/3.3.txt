1. Необходимо расширить функцию переводчика так, чтобы она принимала следующие параметры: 
 - путь к файлу с текстом;
 - путь к файлу с результатом;
 - язык, с которго перевести;
 - язык, на который перевести.
 У вас есть три файла с новостями на разных языках. Функция должна взять каждый файл, перевести,
сохранить результат в новом файле.
